# Dataset Documentation

## Paper Data Assets

The MetServ-VL paper organizes meteorological service data into three asset groups:

| Symbol | Name | Scale In Paper | Description |
| --- | --- | ---: | --- |
| `Dexpert` | Expert-Annotated Data | 70K+ | Image-text pairs annotated or corrected by professional meteorologists. |
| `Daug` | Synthetic-Augmented Data | 30K+ | VLM-generated pseudo labels corrected and verified by humans. |
| `Db` | Benchmark Evaluation Dataset | 1K+ | Expert-annotated benchmark samples for systematic evaluation. |

The full paper data include operational meteorological service products such as precipitation forecasts, high-temperature forecasts, strong-wind products, sandstorms, dense fog, rainstorm warnings, cold-wave warnings, typhoon products, severe convection, snowstorms, and meteorological drought products.

## Public Repository Subset

This repository contains a public benchmark subset and evaluation files:

- **Images**: 109 meteorological service images in `data/dataset/images/`.
- **Evaluation samples**: 470 samples in `data/dataset/labels/vlm_test_dataset.json`.
- **Public image-subset samples**: 109 samples in `data/dataset/labels/vlm_test_dataset1.json`.
- **Caption template**: `data/dataset/image-caption.json`.

Some entries in the 470-sample file refer to images from the full benchmark that are not included in the local public subset. Use `vlm_test_dataset1.json` when you need a one-to-one match with the local `images/` directory.

## Evaluation Entry Format

Each evaluation sample follows the paper format:

```json
{
  "index": 0,
  "image_path": "200000694-图1.jpg",
  "human_answer": "Reference forecast text written by a professional weather forecaster",
  "assistant_answer": "Forecast generated by the VLM being evaluated",
  "question": "请基于提供的预报图给出具体预报"
}
```

Fields:

| Field | Description |
| --- | --- |
| `index` | Sample index. |
| `image_path` | Meteorological service image path or file name. |
| `question` | Natural-language instruction given to the VLM. |
| `human_answer` | Expert reference answer. |
| `assistant_answer` | Model answer to be evaluated. |

## Caption Label Format

Image-caption labels produced by the data construction tools use:

```json
{
  "captions": [
    {
      "role": "caption",
      "content": "气象服务图像对应的专家文本描述"
    }
  ]
}
```

## Task Coverage

The benchmark focuses on visual understanding and text generation for Chinese meteorological services:

- Weather chart captioning.
- Weather forecast generation from professional meteorological images.
- Forecast evaluation against expert references.

## Region And Phenomena Coverage

The LLM-as-Judge protocol includes a region knowledge base for common Chinese meteorological regions:

- 江南, 华南, 淮南, 东北, 江汉, 江淮, 黄淮, 华北.
- 新疆伊犁河谷 and 内蒙古河套地区.

Common weather phenomena include:

- Rainfall: 小雨, 中雨, 大雨, 暴雨, 大暴雨, 特大暴雨.
- Wind: 4-6级风, 阵风.
- Temperature: 高温天气, 35℃以上.
- Severe weather: 短时强降水, 雷暴大风, 冰雹, 台风, 寒潮, 暴雪, 气象干旱.
